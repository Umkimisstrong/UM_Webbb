■■■ Emmet(에밋) ■■■	
  에밋을 익숙하게 사용하면 코딩속도가 빠르다
○ 개요
   
   - 기존에는 Zen Coding(젠코딩) 이라는 이름으로 사용되었다.
   - 이후... Google에서 이 도구를 수용하면서 emmet 이라는 이름으로 변경되었다.
   - HTML 생산성 향상 도구(HTML 코딩을 빠르게 하기 위한 플러그인)
   - CSS 선택자 사용(CSS3 지원)
     → HTML을 CSS 선택자를 통해 검색하는 방법 대신
        CSS 선택자를 통해 HTML 을 생성하도록 하는 개념.

○ 환경 설정

    - 대부분의 툴에서 지원하고 있음
      (※ 그리고 대부분의 편집기 내에서의 액션 키 : Ctrl + E )

    - http://emmet.io
      ① 메인페이지 좌측에 있는 DOWNLOAD 버튼 클릭
         → https://emmet.io/download/ 로 이동
      ② Donwload 리스트들 중 『Eclipse/Aptana』 버튼 클릭
         → https://github.com/emmetio/emmet-eclipse#readme 로 이동
      ③ Installation 항목대로 진행
         ㄱ. 이클립스 상 help ▷ Install New Software 클릭..
         ㄴ. 이 주소 → 『http://download.emmet.io/eclipse/updates/』 를 ADD 하고나서
         ㄷ. Emmet 체크박스를 체크하고 Next - 동의.... 동의.. Finish 하면
             경고창이 뜬다.. Install Anyway 하고 시작
         ㄹ. Install 끝나면 Restart   
      ④ Emmet 플러그인이 제대로 설치되고 이클립스가 정상적으로 재구동 되었다면..
         이클립스의 기본 메뉴에 Emmet 항목이 추가되었음을 확인할 수 있음. 
         ( 단, emmet 에 익숙해지는 동안 이 메뉴를 사용할 일은 특별히 없음 )

           ※ Emmet은 Eclipse에서 사용하는 단축기와 중복됨
